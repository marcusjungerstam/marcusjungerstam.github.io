    FX in game layering

    Transferring effects from the design stage in compositing or 3D applications to a game engine is always a challenging process. For this game, we layered various images, spine animations, and sequences, while fine-tuning the colors through code to achieve a unique look for each symbol. Below, I showcase the layering process we used to create a standard symbol win in the game.
