    I had the chance to work on the intro movie for the slot game Epic Dreams.

    In this project, I handled look development, character grooming, environment modeling, matte painting, environmental effects, effects and compositing.

    Facts:

    We created this intro using assets straight from the game Epic Dreams.

    As my colleagues and I worked on the background and characters, we always envisioned them coming together for some kind of epic intro.
    While the scope of the intro kept shrinking as the game production progressed, we managed to pull it off—mainly because we were able to leverage the existing game assets. It was a fun challenge, and we're thrilled with how it turned out!
